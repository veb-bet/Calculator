# Calculator

Это простое React-приложение-калькулятор. Оно позволяет пользователям выполнять основные арифметические операции, такие как сложение, вычитание, умножение и деление.

## Возможности

- Поддержка ввода цифр
- Поддержка основных арифметических операций
- Поддержка операций с десятичными числами
- Поддержка удаления последнего введенного символа
- Поддержка очистки ввода
- Отображение сообщения об ошибке при неправильном вводе

## Как использовать

1. Склонируйте репозиторий:

```
   git clone https://github.com/veb-bet/Calculator.git
```

2. Перейдите в директорию проекта и установите зависимости:

```
   cd simple-calculator
   npm install
```

3. Запустите сервер разработки:

```
   npm start
```

4. Откройте в браузере `http://localhost:3000` для использования калькулятора.

## Стек технологий
